# Dashboard de Asignaciones - Sales Portfolio

Dashboard interactivo para visualizar asignaciones de portfolio de ventas.

## Características

- Vista diaria (PORTFOLIO_LOG) y mensual (PORTFOLIO)
- Filtros interactivos: País, Producto, Agencia, Criterio, Lista
- Búsqueda por texto en filtros
- Variaciones vs día anterior y mes anterior
- Multi-país: MLA, MLM, MLB
